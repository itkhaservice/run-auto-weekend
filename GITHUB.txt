IV. FILE main.py (PLAYWRIGHT PYTHON)
Nguyên tắc bắt buộc

Luôn dùng headless=True

Không dùng time.sleep() bừa bãi

Ưu tiên wait_for_selector

Code phải chạy KHÔNG cần màn hình

Ví dụ mẫu
from playwright.sync_api import sync_playwright
import pandas as pd

def main():
    df = pd.read_excel("data.xlsx")

    with sync_playwright() as p:
        browser = p.chromium.launch(headless=True)
        page = browser.new_page()

        page.goto("https://example.com")

        for _, row in df.iterrows():
            page.fill("#username", row["username"])
            page.fill("#password", row["password"])
            page.click("#login")
            page.wait_for_timeout(2000)

        browser.close()

if __name__ == "__main__":
    main()


📌 KHÔNG dùng:

launch(headless=False)

Code cần click chuột tay

Code phụ thuộc kích thước màn hình

VIII. FILE GITHUB ACTIONS (run.yml)

📍 Vị trí:

.github/workflows/run.yml

Nội dung đầy đủ
name: Playwright Auto Weekly

on:
  schedule:
    - cron: '0 2 * * 0'   # 02:00 sáng Chủ Nhật (UTC)
  workflow_dispatch:

jobs:
  run-playwright:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout source code
        uses: actions/checkout@v4

      - name: Setup Python
        uses: actions/setup-python@v5
        with:
          python-version: '3.10'

      - name: Install dependencies
        run: |
          pip install -r requirements.txt
          playwright install --with-deps

      - name: Run Playwright script
        run: |
          python main.py

IX. GIẢI THÍCH CRON TIME
0 2 * * 0

Trường	Ý nghĩa
0	phút
2	giờ
*	ngày
*	tháng
0	Chủ Nhật

📌 GitHub dùng UTC
👉 Việt Nam (UTC+7) = 09:00 sáng Chủ Nhật
